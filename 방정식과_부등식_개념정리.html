1단원: 복소수와 그 연산

1-1. 복소수란 무엇일까?


지원아, 수의 세계가 점점 넓어지고 있어!
초등학교 때는 자연수(1, 2, 3...)만 알았고, 중학교에서 음수(-1, -2...)와 무리수($\sqrt{2}$)를 배웠잖아.
그런데 $x^2 = -1$을 만족하는 수는? 실수 중에는 없어! 어떤 수를 제곱해도 양수나 0이 되니까.

그래서 수학자들이 "제곱하면 -1이 되는 새로운 수"를 만들었어. 이걸 $i$라고 불러!
마치 게임에서 새로운 맵이 열리는 것처럼, 수의 세계가 한 단계 더 확장되는 거야.

허수단위 $i$

$i^2 = -1$을 만족하는 수 $i$를 허수단위라고 해.

즉 $i = \sqrt{-1}$이야.

복소수의 뜻

지원아, 임의의 두 실수 $a$, $b$에 대하여

$a + bi$

꼴로 나타내어지는 수를 복소수라고 해!

  • $a$: 실수부분 (real part)
  • $b$: 허수부분 (imaginary part)

지원아, 이것만 기억해!

복소수 $a+bi$ ($a$, $b$는 실수)

$b=0$이면 → 실수 ($a$)

$b \neq 0$이면 → 허수

실수도 복소수야! $a = a + 0i$로 쓸 수 있으니까!


$-3$도 복소수야! $-3 = (-3) + 0 \times i$로 생각하면 돼.
$3+2i$에서 실수부분은 3, 허수부분은 2야.
$-2i$에서 실수부분은 0, 허수부분은 -2야.
$2\sqrt{5}$에서 실수부분은 $2\sqrt{5}$, 허수부분은 0이야 (이건 실수!).

문제 1: 다음 복소수의 실수부분과 허수부분을 구하시오.

(1) $2+\sqrt{3}i$    (2) $3-i$    (3) $-2i$    (4) $2\sqrt{5}$

지원아, $a+bi$ 꼴에서 $a$가 실수부분, $b$가 허수부분이야!

(1) $2+\sqrt{3}i$: 실수부분 = $2$, 허수부분 = $\sqrt{3}$
(2) $3-i = 3+(-1)i$: 실수부분 = $3$, 허수부분 = $-1$
(3) $-2i = 0+(-2)i$: 실수부분 = $0$, 허수부분 = $-2$
(4) $2\sqrt{5} = 2\sqrt{5}+0i$: 실수부분 = $2\sqrt{5}$, 허수부분 = $0$
답: (1) 실수부분 2, 허수부분 $\sqrt{3}$ (2) 실수부분 3, 허수부분 $-1$ (3) 실수부분 0, 허수부분 $-2$ (4) 실수부분 $2\sqrt{5}$, 허수부분 0

1-2. 서로 같은 복소수


지원아, 좌표를 생각해봐!
점 $(3, 5)$와 점 $(3, 5)$이 같으려면 $x$좌표도 같고 $y$좌표도 같아야 하잖아?
복소수도 마찬가지야! $a+bi = c+di$가 되려면 실수부분끼리, 허수부분끼리 각각 같아야 해!

지원아, 이것만 기억해!

두 복소수 $a+bi$, $c+di$ ($a, b, c, d$는 실수)가 서로 같으면

$a+bi = c+di$   $\Leftrightarrow$   $a=c$이고 $b=d$

특히 $a+bi = 0$이면 $a=0$, $b=0$

예제 1: 다음 등식을 만족시키는 실수 $x$, $y$의 값을 구하시오.

$(x-2y)+(3x+y)i = 4+5i$

지원아, 실수부분끼리, 허수부분끼리 비교하면 돼!

두 복소수가 서로 같으려면 실수부분과 허수부분이 각각 같아야 하므로
$x - 2y = 4$   ... ①
$3x + y = 5$   ... ②

①, ②을 연립하여 풀면
$x = 2$, $y = -1$
답: $x=2$, $y=-1$

문제 2: 다음 등식을 만족시키는 실수 $x$, $y$의 값을 구하시오.

(1) $x+yi = 1-5i$    (2) $2x-yi = 4+\sqrt{6}i$

지원아, 실수부분=실수부분, 허수부분=허수부분으로 비교해!

(1) $x = 1$, $y = -5$

(2) $2x = 4$이므로 $x = 2$, $-y = \sqrt{6}$이므로 $y = -\sqrt{6}$
답: (1) $x=1$, $y=-5$   (2) $x=2$, $y=-\sqrt{6}$

문제 3: 다음 등식을 만족시키는 실수 $x$, $y$의 값을 구하시오.

(1) $(2x-1)+(y-2)i = 3+i$    (2) $(x+y+2)-(3x+y)i = 0$

(1) 지원아, 실수부분과 허수부분을 비교하자!

$2x-1 = 3$ → $x = 2$
$y-2 = 1$ → $y = 3$

(2) 지원아, 오른쪽이 0이면 $0+0i$라는 뜻이야!

$x+y+2 = 0$   ... ①
$-(3x+y) = 0$, 즉 $3x+y = 0$   ... ②
①-②: $-2x+2 = 0$ → $x = 1$
②에서 $y = -3$
답: (1) $x=2$, $y=3$   (2) $x=1$, $y=-3$

1-3. 켤레복소수


지원아, 거울을 생각해봐!
거울에 비친 나는 좌우가 반대이듯, 켤레복소수는 허수부분의 부호만 반대인 짝꿍이야!
$3+2i$의 거울 속 모습 → $3-2i$ (허수부분 부호만 뒤집기!)

지원아, 이것만 기억해!

복소수 $a+bi$의 켤레복소수: $\overline{a+bi} = a-bi$

또한 $\overline{a-bi} = a+bi$

두 복소수 $a+bi$와 $a-bi$는 서로 켤레복소수!


켤레복소수가 자기 자신과 같은 수는? $\overline{z} = z$이면 허수부분이 0, 즉 실수야!
예: $\overline{5} = 5$, $\overline{-3} = -3$ (실수의 켤레복소수는 자기 자신!)

문제 4: 다음 복소수의 켤레복소수를 구하시오.

(1) $3+2i$    (2) $3-2i$    (3) $-3i$    (4) $1+\sqrt{3}$

지원아, 허수부분의 부호만 뒤집으면 돼!

(1) $\overline{3+2i} = 3-2i$
(2) $\overline{3-2i} = 3+2i$
(3) $\overline{-3i} = \overline{0+(-3)i} = 0+3i = 3i$
(4) $1+\sqrt{3}$은 실수이므로 켤레복소수는 자기 자신: $1+\sqrt{3}$
답: (1) $3-2i$ (2) $3+2i$ (3) $3i$ (4) $1+\sqrt{3}$

1-4. 복소수의 사칙연산


지원아, 복소수 계산은 문자 계산이랑 똑같아!
$i$를 그냥 문자 $x$처럼 생각하고 계산하면 돼. 단 하나 다른 점은 $i^2$이 나오면 $-1$로 바꿔주는 것뿐이야!
$(2+3x) + (1+5x) = 3+8x$ 처럼 → $(2+3i) + (1+5i) = 3+8i$ 이렇게!

덧셈과 뺄셈

복소수의 덧셈과 뺄셈

$a, b, c, d$가 실수일 때

덧셈: $(a+bi)+(c+di) = (a+c)+(b+d)i$

뺄셈: $(a+bi)-(c+di) = (a-c)+(b-d)i$

실수부분끼리, 허수부분끼리 계산!

문제 5: 다음을 계산하시오.

(1) $(3+2i)+(2-3i)$    (2) $(3-3i)-(-2+i)$    (3) $(2+\sqrt{7}i)-(2-\sqrt{7}i)$

지원아, 실수부분끼리, 허수부분끼리 각각 계산하면 돼!

(1) $(3+2)+(2-3)i = 5-i$

(2) $(3-(-2))+(-3-1)i = 5-4i$

(3) $(2-2)+(\sqrt{7}-(-\sqrt{7}))i = 0+2\sqrt{7}i = 2\sqrt{7}i$
답: (1) $5-i$   (2) $5-4i$   (3) $2\sqrt{7}i$

곱셈

복소수의 곱셈

$a, b, c, d$가 실수일 때

$(a+bi)(c+di) = (ac-bd)+(ad+bc)i$

$i$를 문자처럼 전개하고, $i^2 = -1$을 대입!

예제 2: 다음을 계산하시오.

(1) $(2+3i)(4+i)$    (2) $(3+i)(3-i)$

지원아, 분배법칙으로 전개하고 $i^2=-1$을 이용하면 돼!

(1) $(2+3i)(4+i)$
$= 8+2i+12i+3i^2$
$= 8+(2+12)i+3\times(-1)$
$= 5+14i$
(2) $(3+i)(3-i)$
$= 9-3i+3i-i^2$
$= 9+(-3+3)i-(-1)$
$= 10$
답: (1) $5+14i$   (2) $10$

$(a+bi)(a-bi) = a^2+b^2$ (항상 실수가 돼!)
이건 켤레복소수끼리 곱하면 항상 실수가 된다는 뜻이야. 나눗셈에서 엄청 유용해!

문제 6: 다음을 계산하시오.

(1) $(2+i)(1-3i)$    (2) $(1+2i)(1-2i)$    (3) $(\sqrt{3}+\sqrt{2}i)(\sqrt{3}-\sqrt{2}i)$

(1) $= 2-6i+i-3i^2 = 2-5i-3(-1) = 5-5i$

(2) 켤레복소수끼리의 곱! $= 1^2+(2)^2 = 1+4 = 5$

(3) 역시 켤레복소수! $= (\sqrt{3})^2+(\sqrt{2})^2 = 3+2 = 5$
답: (1) $5-5i$   (2) $5$   (3) $5$

문제 7: 허수단위 $i$의 거듭제곱 $i^3, i^4, i^5, \ldots, i^{12}$의 값을 구하고, $i^n$의 값의 규칙성을 말하시오. (단, $n$은 자연수)

지원아, $i$의 거듭제곱은 4개씩 반복돼! 한번 차근차근 계산해보자.

$i^1 = i$
$i^2 = -1$
$i^3 = i^2 \times i = -1 \times i = -i$
$i^4 = i^2 \times i^2 = (-1)(-1) = 1$
$i^5 = i^4 \times i = 1 \times i = i$
$i^6 = i^4 \times i^2 = 1 \times (-1) = -1$
... 이렇게 $i, -1, -i, 1$이 4개씩 반복돼!
답: $i^n$의 값은 $n$을 4로 나눈 나머지에 따라 결정. 나머지가 1→$i$, 2→$-1$, 3→$-i$, 0→$1$

나눗셈

복소수의 나눗셈

$a, b, c, d$가 실수이고 $c+di \neq 0$일 때

$\dfrac{a+bi}{c+di} = \dfrac{(a+bi)(c-di)}{(c+di)(c-di)} = \dfrac{ac+bd}{c^2+d^2}+\dfrac{bc-ad}{c^2+d^2}i$

분모의 켤레복소수를 분모, 분자에 각각 곱해서 분모를 실수로 만들어!


지원아, 분모의 유리화 기억나지? $\frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$ 이렇게!
복소수 나눗셈도 똑같은 원리야. 분모에 허수가 있으면, 분모의 켤레복소수를 곱해서 분모를 실수로 바꿔주는 거야!

예제 3: $\dfrac{3+4i}{2-i}$를 계산하여 $a+bi$ 꼴로 나타내시오. (단, $a$, $b$는 실수)

지원아, 분모의 켤레복소수 $2+i$를 위아래에 곱하자!

$\dfrac{3+4i}{2-i} = \dfrac{(3+4i)(2+i)}{(2-i)(2+i)}$

분자: $(3+4i)(2+i) = 6+3i+8i+4i^2 = 6+11i-4 = 2+11i$
분모: $(2-i)(2+i) = 4-i^2 = 4+1 = 5$

$= \dfrac{2+11i}{5} = \dfrac{2}{5}+\dfrac{11}{5}i$
답: $\dfrac{2}{5}+\dfrac{11}{5}i$

문제 8: 다음을 계산하여 $a+bi$ 꼴로 나타내시오. (단, $a$, $b$는 실수)

(1) $\dfrac{1}{i}$    (2) $\dfrac{1}{\sqrt{3}-i}$    (3) $\dfrac{2\sqrt{2}+i}{2\sqrt{2}-i}$

지원아, 모두 분모의 켤레복소수를 곱해서 분모를 실수로 만들자!

(1) $\dfrac{1}{i} = \dfrac{1 \times (-i)}{i \times (-i)} = \dfrac{-i}{-i^2} = \dfrac{-i}{1} = -i$

(2) $\dfrac{1}{\sqrt{3}-i} = \dfrac{\sqrt{3}+i}{(\sqrt{3}-i)(\sqrt{3}+i)} = \dfrac{\sqrt{3}+i}{3+1} = \dfrac{\sqrt{3}}{4}+\dfrac{1}{4}i$

(3) $\dfrac{2\sqrt{2}+i}{2\sqrt{2}-i} = \dfrac{(2\sqrt{2}+i)^2}{(2\sqrt{2}-i)(2\sqrt{2}+i)} = \dfrac{8+4\sqrt{2}i+i^2}{8+1} = \dfrac{7+4\sqrt{2}i}{9} = \dfrac{7}{9}+\dfrac{4\sqrt{2}}{9}i$
답: (1) $-i$   (2) $\dfrac{\sqrt{3}}{4}+\dfrac{1}{4}i$   (3) $\dfrac{7}{9}+\dfrac{4\sqrt{2}}{9}i$

1-5. 음수의 제곱근


지원아, 중학교 때 $\sqrt{4} = 2$는 "제곱하면 4가 되는 양수"였잖아?
그런데 $\sqrt{-4}$는? 실수에서는 불가능했는데, 이제 $i$를 배웠으니까 할 수 있어!
$\sqrt{-4} = \sqrt{4} \times \sqrt{-1} = 2i$! 신기하지?

지원아, 이것만 기억해!

$a > 0$일 때

$-a$의 제곱근은 $\sqrt{a}i$와 $-\sqrt{a}i$

$\sqrt{-a} = \sqrt{a}i$


주의! 음수의 제곱근은 반드시 $\sqrt{a}i$ 꼴로 바꾼 다음에 계산해야 해!
$\sqrt{-5}\sqrt{-5} = \sqrt{5}i \times \sqrt{5}i = 5i^2 = -5$ (O)
$\sqrt{-5}\sqrt{-5} = \sqrt{(-5)(-5)} = \sqrt{25} = 5$ (X) ← 이렇게 하면 안 돼!
$\sqrt{a}\sqrt{b} = \sqrt{ab}$는 $a \geq 0$, $b \geq 0$일 때만 성립해!

문제 9: 다음 수의 제곱근을 구하시오.

(1) $-25$    (2) $-64$    (3) $-54$

(1) $-25$의 제곱근: $\pm\sqrt{25}i = \pm 5i$

(2) $-64$의 제곱근: $\pm\sqrt{64}i = \pm 8i$

(3) $-54$의 제곱근: $\pm\sqrt{54}i = \pm 3\sqrt{6}i$
답: (1) $\pm 5i$   (2) $\pm 8i$   (3) $\pm 3\sqrt{6}i$

문제 10: 다음을 계산하여 $a+bi$ 꼴로 나타내시오. (단, $a$, $b$는 실수)

(1) $\sqrt{-2}+\sqrt{-8}$    (2) $\sqrt{3}\sqrt{-27}$    (3) $\dfrac{\sqrt{-45}}{\sqrt{5}}$

지원아, 먼저 $\sqrt{-a} = \sqrt{a}i$로 바꾸고 나서 계산해!

(1) $\sqrt{-2}+\sqrt{-8} = \sqrt{2}i + 2\sqrt{2}i = 3\sqrt{2}i$

(2) $\sqrt{3}\sqrt{-27} = \sqrt{3} \times 3\sqrt{3}i = 3 \times 3 \times i = 9i$

(3) $\dfrac{\sqrt{-45}}{\sqrt{5}} = \dfrac{3\sqrt{5}i}{\sqrt{5}} = 3i$
답: (1) $3\sqrt{2}i$   (2) $9i$   (3) $3i$

스스로 확인하기

확인 1: $\alpha=3+2i$, $\beta=3-2i$일 때, 다음을 계산하여 $a+bi$ 꼴로 나타내시오.

(1) $\alpha+\beta$    (2) $\alpha\beta$    (3) $\dfrac{\alpha}{\beta}$    (4) $\alpha\bar{\alpha}+\beta\bar{\beta}$

(1) $\alpha+\beta = (3+2i)+(3-2i) = 6$

(2) $\alpha\beta = (3+2i)(3-2i) = 9-4i^2 = 9+4 = 13$

(3) $\dfrac{\alpha}{\beta} = \dfrac{3+2i}{3-2i} = \dfrac{(3+2i)^2}{(3-2i)(3+2i)} = \dfrac{9+12i+4i^2}{13} = \dfrac{5+12i}{13} = \dfrac{5}{13}+\dfrac{12}{13}i$

(4) $\bar{\alpha} = 3-2i = \beta$, $\bar{\beta} = 3+2i = \alpha$이므로
$\alpha\bar{\alpha}+\beta\bar{\beta} = \alpha\beta + \beta\alpha = 2\alpha\beta = 2 \times 13 = 26$
답: (1) $6$ (2) $13$ (3) $\dfrac{5}{13}+\dfrac{12}{13}i$ (4) $26$

확인 2: 다음을 계산하여 $a+bi$ 꼴로 나타내시오.

2-1. 이차방정식의 실근과 허근


지원아, 이차방정식의 근(답)을 구할 때 근의 공식을 쓰잖아?
그런데 루트 안에 음수가 들어가면 어떡해? 예전에는 "해 없음"이었는데,
이제 $i$를 배웠으니까 허수인 근(허근)도 구할 수 있어!
실수인 근은 실근, 허수인 근은 허근이라고 불러.

지원아, 이것만 기억해!

이차방정식 $ax^2+bx+c=0$의 근:

$x = \dfrac{-b \pm \sqrt{b^2-4ac}}{2a}$

$b^2-4ac \geq 0$ → 근은 실근

$b^2-4ac < 0$ → 근은 허근

예제 1: 다음 이차방정식을 풀고, 그 근이 실근인지 허근인지 말하시오.

(1) $x^2-4x+2=0$    (2) $2x^2+3x+2=0$

지원아, 근의 공식에 대입해서 풀어보자!

(1) $a=1$, $b=-4$, $c=2$
$x = \dfrac{-(-4) \pm \sqrt{(-4)^2-4 \times 1 \times 2}}{2 \times 1} = \dfrac{4 \pm \sqrt{8}}{2} = 2 \pm \sqrt{2}$
루트 안이 양수 → 실근
(2) $a=2$, $b=3$, $c=2$
$x = \dfrac{-3 \pm \sqrt{9-16}}{4} = \dfrac{-3 \pm \sqrt{-7}}{4} = \dfrac{-3 \pm \sqrt{7}i}{4}$
루트 안이 음수 → 허근
답: (1) $x=2\pm\sqrt{2}$, 실근   (2) $x=\dfrac{-3\pm\sqrt{7}i}{4}$, 허근

이차방정식이 서로 다른 두 허근을 가지면, 두 근은 항상 서로 켤레복소수야!
위 (2)번에서 $\dfrac{-3+\sqrt{7}i}{4}$와 $\dfrac{-3-\sqrt{7}i}{4}$는 켤레복소수 관계!

문제 1: 다음 이차방정식을 풀고, 그 근이 실근인지 허근인지 말하시오.

(3) $x^2+x-3=0$    (4) $2x^2+4x+3=0$

(1) $x = \dfrac{2\pm\sqrt{4+8}}{2} = \dfrac{2\pm\sqrt{12}}{2} = 1\pm\sqrt{3}$ → 실근

(2) $x = \dfrac{1\pm\sqrt{1-8}}{4} = \dfrac{1\pm\sqrt{-7}}{4} = \dfrac{1\pm\sqrt{7}i}{4}$ → 허근

(3) $x = \dfrac{-1\pm\sqrt{1+12}}{2} = \dfrac{-1\pm\sqrt{13}}{2}$ → 실근

(4) $x = \dfrac{-4\pm\sqrt{16-24}}{4} = \dfrac{-4\pm\sqrt{-8}}{4} = \dfrac{-4\pm2\sqrt{2}i}{4} = \dfrac{-2\pm\sqrt{2}i}{2}$ → 허근
답: (1) $1\pm\sqrt{3}$, 실근 (2) $\dfrac{1\pm\sqrt{7}i}{4}$, 허근 (3) $\dfrac{-1\pm\sqrt{13}}{2}$, 실근 (4) $\dfrac{-2\pm\sqrt{2}i}{2}$, 허근

2-2. 판별식


지원아, 매번 근의 공식을 다 계산하지 않고도 "이 방정식의 근이 실수야? 허수야?"를 알아내는 방법이 있어!
바로 루트 안의 식 $b^2-4ac$만 보면 돼! 이걸 판별식이라 부르고 $D$로 써.
마치 날씨 앱에서 비 올 확률만 보고 우산을 가져갈지 결정하는 것처럼!

지원아, 이것만 기억해! - 이차방정식의 근의 판별

이차방정식 $ax^2+bx+c=0$에서 $D = b^2-4ac$라 할 때

$D > 0$ → 서로 다른 두 실근

$D = 0$ → 중근(실근)

$D < 0$ → 서로 다른 두 허근


$b$가 짝수일 때 $ax^2+2b'x+c=0$ 꼴이면 $\dfrac{D}{4} = b'^2-ac$를 사용하면 계산이 더 간단해!
판별 결과는 동일하니까 편한 걸로 쓰면 돼.

문제 2: 다음 이차방정식의 근을 판별하시오.

(1) $x^2-5x-2=0$    (2) $x^2+4x+4=0$

(3) $2x^2-x+3=0$    (4) $-3x^2+2x+2=0$

지원아, $D=b^2-4ac$를 계산해서 부호만 확인하면 돼!

(1) $D = 25-4(1)(-2) = 25+8 = 33 > 0$ → 서로 다른 두 실근

(2) $D = 16-4(1)(4) = 16-16 = 0$ → 중근

(3) $D = 1-4(2)(3) = 1-24 = -23 < 0$ → 서로 다른 두 허근

(4) $D = 4-4(-3)(2) = 4+24 = 28 > 0$ → 서로 다른 두 실근
답: (1) 서로 다른 두 실근 (2) 중근 (3) 서로 다른 두 허근 (4) 서로 다른 두 실근

예제 2: 이차방정식 $x^2-6x+7-k=0$이 서로 다른 두 실근을 가질 때, 실수 $k$의 값의 범위를 구하시오.

지원아, "서로 다른 두 실근" → $D > 0$이야!

$\dfrac{D}{4} = (-3)^2-(7-k) = 9-7+k = k+2$

서로 다른 두 실근을 가지려면 $\dfrac{D}{4} > 0$이어야 하므로
$k+2 > 0$, 즉 $k > -2$
답: $k > -2$

문제 3: 이차방정식 $x^2+3x+k+3=0$이 다음과 같은 근을 가질 때, 실수 $k$의 값 또는 그 범위를 구하시오.

(1) 서로 다른 두 실근    (2) 중근    (3) 서로 다른 두 허근

$D = 9-4(k+3) = 9-4k-12 = -3-4k$

(1) 서로 다른 두 실근: $D > 0$
$-3-4k > 0$ → $-4k > 3$ → $k < -\dfrac{3}{4}$

$-3-4k = 0$ → $k = -\dfrac{3}{4}$

(3) 서로 다른 두 허근: $D < 0$
$-3-4k < 0$ → $k > -\dfrac{3}{4}$

3-1. 근과 계수의 관계


지원아, 신기한 거 알려줄까?
이차방정식의 두 근을 직접 구하지 않아도, 두 근의 합과 곱을 바로 알 수 있어!
마치 선물 상자를 열어보지 않고도 무게로 안에 뭐가 있는지 추측하는 것처럼,
방정식의 계수(숫자들)만 보면 근의 합과 곱을 알 수 있거든!

지원아, 이것만 기억해! - 근과 계수의 관계

이차방정식 $ax^2+bx+c=0$의 두 근을 $\alpha$, $\beta$라 하면

$\alpha+\beta = -\dfrac{b}{a}$     $\alpha\beta = \dfrac{c}{a}$

두 근의 합 = $-\dfrac{x의\;계수}{x^2의\;계수}$   두 근의 곱 = $\dfrac{상수항}{x^2의\;계수}$


합에서 마이너스 부호 빼먹지 마! $\alpha+\beta = \color{red}{-}\dfrac{b}{a}$
예: $3x^2+2x+1=0$에서 두 근의 합 = $-\dfrac{2}{3}$, 두 근의 곱 = $\dfrac{1}{3}$

문제 1: 다음 이차방정식의 두 근의 합과 곱을 구하시오.

(1) $x^2-4x+5=0$    (2) $2x^2+3x-4=0$

(1) $\alpha+\beta = -\dfrac{-4}{1} = 4$, $\alpha\beta = \dfrac{5}{1} = 5$

(2) $\alpha+\beta = -\dfrac{3}{2}$, $\alpha\beta = \dfrac{-4}{2} = -2$
답: (1) 합: 4, 곱: 5   (2) 합: $-\dfrac{3}{2}$, 곱: $-2$

예제 1: 이차방정식 $x^2-5x-3=0$의 두 근을 $\alpha$, $\beta$라 할 때, $\alpha^2+\beta^2$의 값을 구하시오.

지원아, $\alpha^2+\beta^2$을 $(\alpha+\beta)$와 $\alpha\beta$로 바꿔 쓸 수 있어!

근과 계수의 관계에서 $\alpha+\beta = 5$, $\alpha\beta = -3$

$\alpha^2+\beta^2 = (\alpha+\beta)^2 - 2\alpha\beta$
$= 5^2 - 2 \times (-3)$
$= 25 + 6 = 31$
답: $31$

문제 2: 이차방정식 $2x^2-4x+1=0$의 두 근을 $\alpha$, $\beta$라 할 때, 다음 식의 값을 구하시오.

(1) $\alpha^2+\beta^2$    (2) $(\alpha-\beta)^2$    (3) $\dfrac{\beta}{\alpha}+\dfrac{\alpha}{\beta}$    (4) $(\alpha+1)(\beta+1)$

지원아, 먼저 합과 곱부터 구하자! $\alpha+\beta = 2$, $\alpha\beta = \dfrac{1}{2}$

(1) $\alpha^2+\beta^2 = (\alpha+\beta)^2-2\alpha\beta = 4-1 = 3$

(2) $(\alpha-\beta)^2 = (\alpha+\beta)^2-4\alpha\beta = 4-2 = 2$

(3) $\dfrac{\beta}{\alpha}+\dfrac{\alpha}{\beta} = \dfrac{\alpha^2+\beta^2}{\alpha\beta} = \dfrac{3}{1/2} = 6$

(4) $(\alpha+1)(\beta+1) = \alpha\beta+\alpha+\beta+1 = \dfrac{1}{2}+2+1 = \dfrac{7}{2}$
답: (1) $3$ (2) $2$ (3) $6$ (4) $\dfrac{7}{2}$

3-2. 두 수를 근으로 하는 이차방정식


지원아, 지금까지는 방정식 → 근을 구했는데, 이번에는 반대야!
근 → 방정식을 만드는 거야. 레시피를 보고 요리하는 것(근 구하기)에서,
완성된 요리를 보고 레시피를 만드는 것(방정식 만들기)으로 바뀐 거지!

지원아, 이것만 기억해!

두 수 $\alpha$, $\beta$를 근으로 하고 $x^2$의 계수가 1인 이차방정식:

$x^2 - (\alpha+\beta)x + \alpha\beta = 0$

(두 근의 합)을 빼고, (두 근의 곱)을 더한다!

문제 3: 다음 두 수를 근으로 하고 $x^2$의 계수가 1인 이차방정식을 구하시오.

(1) 합: $(1+\sqrt{2})+(1-\sqrt{2}) = 2$
곱: $(1+\sqrt{2})(1-\sqrt{2}) = 1-2 = -1$
$x^2 - 2x + (-1) = 0$ → $x^2-2x-1=0$

(2) 합: $\dfrac{1+\sqrt{3}i}{2}+\dfrac{1-\sqrt{3}i}{2} = 1$
곱: $\dfrac{(1+\sqrt{3}i)(1-\sqrt{3}i)}{4} = \dfrac{1+3}{4} = 1$
$x^2 - x + 1 = 0$
답: (1) $x^2-2x-1=0$   (2) $x^2-x+1=0$

(2) $t^2-3t-5=0$ → $t = \dfrac{3\pm\sqrt{29}}{2}$
$\alpha = \dfrac{3-\sqrt{29}}{2}$, $\beta = \dfrac{3+\sqrt{29}}{2}$
답: (1) $\alpha=2-\sqrt{2}$, $\beta=2+\sqrt{2}$   (2) $\alpha=\dfrac{3-\sqrt{29}}{2}$, $\beta=\dfrac{3+\sqrt{29}}{2}$
4단원: 이차방정식과 이차함수의 관계

4-1. 이차함수의 그래프와 x축


지원아, 공을 위로 던지면 포물선을 그리면서 올라갔다 내려오잖아?
이 포물선이 바로 이차함수의 그래프야! 그리고 이 포물선이 땅(x축)과 만나는 점이
바로 이차방정식의 근(해)이야! 포물선과 땅의 관계 = 이차함수와 이차방정식의 관계!

지원아, 이것만 기억해!

이차함수 $y=ax^2+bx+c$의 그래프와 $x$축의 교점의 $x$좌표

= 이차방정식 $ax^2+bx+c=0$의 실근

판별식 $D=b^2-4ac$에 따라:

$D > 0$: 서로 다른 두 점에서 만남

$D = 0$: 한 점에서 만남 (접함)

$D < 0$: 만나지 않음

문제 1: 다음 이차함수의 그래프와 $x$축의 위치 관계를 말하시오.

(1) $y=x^2+3x-4$    (2) $y=-2x^2+x-5$

(3) $y=9x^2-6x+1$    (4) $y=2x^2+5x+3$

(1) $D = 9+16 = 25 > 0$ → 두 점에서 만난다

(2) $D = 1-40 = -39 < 0$ → 만나지 않는다

(3) $D = 36-36 = 0$ → 한 점에서 만난다(접한다)

(4) $D = 25-24 = 1 > 0$ → 두 점에서 만난다

문제 2: 이차함수 $y=x^2-3x+k$의 그래프와 $x$축이 서로 다른 두 점에서 만날 때, 실수 $k$의 값의 범위를 구하시오.

$D = 9-4k > 0$
$k < \dfrac{9}{4}$
답: $k < \dfrac{9}{4}$

4-2. 이차함수의 그래프와 직선의 위치 관계


지원아, 포물선과 직선이 만나는 점을 구하는 건, 두 식을 연립방정식으로 푸는 것과 같아!
$y=ax^2+bx+c$와 $y=mx+n$을 같다고 놓으면
$ax^2+(b-m)x+(c-n)=0$이라는 이차방정식이 나오고,
이 방정식의 판별식으로 만나는지 안 만나는지 알 수 있어!

이차함수의 그래프와 직선의 위치 관계

$y=ax^2+bx+c$와 $y=mx+n$의 교점

→ $ax^2+(b-m)x+(c-n)=0$의 판별식 $D$로 판별

$D > 0$: 서로 다른 두 점에서 만남

$D = 0$: 한 점에서 만남 (접함)

$D < 0$: 만나지 않음

예제 1: 이차함수 $y=x^2+4x+3$의 그래프와 직선 $y=3x+k$가 한 점에서 만날 때, 실수 $k$의 값을 구하시오.

지원아, 두 식을 같다고 놓고 판별식을 쓰자!

$x^2+4x+3 = 3x+k$
$x^2+x+3-k = 0$

한 점에서 만남 → $D = 0$
$D = 1-4(3-k) = 1-12+4k = 4k-11 = 0$
$k = \dfrac{11}{4}$
답: $k = \dfrac{11}{4}$

문제 3: 이차함수 $y=x^2-2x+3$의 그래프와 다음 직선의 위치 관계를 말하시오.

(1) $y=-x+1$    (2) $y=x+6$    (3) $y=2x-1$

(1) $x^2-2x+3 = -x+1$ → $x^2-x+2=0$
$D = 1-8 = -7 < 0$ → 만나지 않는다

(2) $x^2-2x+3 = x+6$ → $x^2-3x-3=0$
$D = 9+12 = 21 > 0$ → 두 점에서 만난다

(3) $x^2-2x+3 = 2x-1$ → $x^2-4x+4=0$
$D = 16-16 = 0$ → 한 점에서 만난다(접한다)

문제 4: 이차함수 $y=x^2-2x+4$의 그래프와 직선 $y=4x+k$가 서로 다른 두 점에서 만날 때, 실수 $k$의 값의 범위를 구하시오.

$x^2-2x+4 = 4x+k$ → $x^2-6x+4-k=0$
$D = 36-4(4-k) = 36-16+4k = 20+4k > 0$
$k > -5$
답: $k > -5$

문제 5: 이차함수 $y=-2x^2+x+k$의 그래프와 직선 $y=-5x+3$이 접할 때, 실수 $k$의 값을 구하시오.

$-2x^2+x+k = -5x+3$ → $-2x^2+6x+k-3=0$
$2x^2-6x-k+3=0$
접함 → $D = 0$
$D = 36-4(2)(-k+3) = 36+8k-24 = 12+8k = 0$
$k = -\dfrac{3}{2}$
답: $k = -\dfrac{3}{2}$

스스로 확인하기

확인 1: 이차함수 $y=x^2+4x+a-3$의 그래프가 $x$축과 접할 때, 실수 $a$의 값을 구하시오.

$D = 16-4(a-3) = 16-4a+12 = 28-4a = 0$
$a = 7$
답: $a = 7$

확인 2: 이차함수 $y=2x^2+4x-4$의 그래프와 직선 $y=x+k$가 만나지 않도록 하는 실수 $k$의 값의 범위를 구하시오.

$2x^2+4x-4 = x+k$ → $2x^2+3x-4-k=0$
만나지 않음 → $D < 0$
$D = 9-4(2)(-4-k) = 9+32+8k = 41+8k < 0$
$k < -\dfrac{41}{8}$
답: $k < -\dfrac{41}{8}$
5단원: 이차함수의 최대, 최소

5-1. 제한된 범위에서 최댓값과 최솟값


지원아, 놀이공원에서 롤러코스터를 타면 올라갔다가 내려오잖아?
그런데 "출발점부터 꼭대기까지만 본다" vs "전체 트랙을 본다"에 따라 가장 높은 곳이 달라질 수 있어!
이차함수도 마찬가지야. $x$의 범위가 제한되면, 최댓값/최솟값이 달라질 수 있어!
핵심은 꼭짓점이 범위 안에 있느냐 없느냐야!

지원아, 이것만 기억해!

$f(x) = a(x-p)^2+q$에서 $a \leq x \leq \beta$일 때

꼭짓점($x=p$)이 범위 안에 있으면:

$a > 0$: 최솟값 = $q$ (꼭짓점), 최댓값 = $f(\alpha)$, $f(\beta)$ 중 큰 값

$a < 0$: 최댓값 = $q$ (꼭짓점), 최솟값 = $f(\alpha)$, $f(\beta)$ 중 작은 값

꼭짓점이 범위 밖이면:

$f(\alpha)$, $f(\beta)$ 중 큰 값이 최댓값, 작은 값이 최솟값


3단계로 풀자!
1단계: 꼭짓점 좌표 구하기 (완전제곱식으로 변환)
2단계: 꼭짓점이 범위 안에 있는지 확인
3단계: 양 끝점과 꼭짓점의 함숫값 비교!

예제 1: $0 \leq x \leq 3$일 때, 이차함수 $y=x^2-4x+3$의 최댓값과 최솟값을 구하시오.

지원아, 먼저 꼭짓점을 구하자!

$y = x^2-4x+3 = (x-2)^2-1$
꼭짓점: $(2, -1)$ → $x=2$는 $0 \leq x \leq 3$ 범위 안에 있어!

$x=0$일 때 $y=3$
$x=2$일 때 $y=-1$ (꼭짓점)
$x=3$일 때 $y=0$

$a > 0$이므로 아래로 볼록 → 꼭짓점에서 최솟값
답: 최댓값: $3$ (x=0), 최솟값: $-1$ (x=2)

문제 1: 다음 주어진 $x$의 값의 범위에서 이차함수의 최댓값과 최솟값을 구하시오.

(1) $y=x^2-4x$ ($1 \leq x \leq 3$)    (2) $y=-3x^2+6x+2$ ($-3 \leq x \leq -1$)

(1) $y = (x-2)^2-4$, 꼭짓점 $(2, -4)$ → 범위 안!
$f(1) = 1-4 = -3$, $f(2) = -4$, $f(3) = 9-12 = -3$
최댓값: $-3$, 최솟값: $-4$

(2) $y = -3(x-1)^2+5$, 꼭짓점 $(1, 5)$ → 범위 밖!
$f(-3) = -3(9)+6(-3)+2 = -27-18+2 = -43$
$f(-1) = -3(1)+6(-1)+2 = -3-6+2 = -7$
최댓값: $-7$, 최솟값: $-43$
답: (1) 최댓값: $-3$, 최솟값: $-4$   (2) 최댓값: $-7$, 최솟값: $-43$

5-2. 이차함수의 최대, 최소의 활용


지원아, 이차함수의 최대/최소는 실생활에서 엄청 많이 쓰여!
"울타리로 만들 수 있는 가장 넓은 공간은?" "공을 던졌을 때 가장 높이 올라가는 높이는?"
이런 문제들이 다 이차함수의 최대/최소 문제야!
핵심은 변수를 정하고 → 이차식으로 만들고 → 꼭짓점을 찾는 것!

예제 2: 길이가 10m인 울타리로 벽면에 직사각형 모양의 공간을 만들 때, 바닥의 최대 넓이를 구하시오. (단, 울타리 두께 무시)

지원아, 먼저 변수를 정하자!

변수 정하기: 가로 = $x$m, 세로 = $y$m²

변수의 범위: 가로의 길이가 $x$m이면 세로의 길이는 $(10-x)/2$가 아니라...
벽면이 한쪽이므로 울타리: $x + 2y = 10$에서 $y = \dfrac{10-x}{2}$
$x > 0$, $10-x > 0$이므로 $0 < x < 10$

관계식 세우기: 넓이 $= x \cdot \dfrac{10-x}{2} = \dfrac{x(10-x)}{2}$
$= \dfrac{-x^2+10x}{2} = -\dfrac{1}{2}(x-5)^2+\dfrac{25}{2}$

답 구하기: $x=5$일 때 최댓값 $\dfrac{25}{2} = 12.5$m²
답: 바닥의 최대 넓이는 $\dfrac{25}{2}$m² (= 12.5m²)

문제 2: 10m 높이의 발판에서 다이빙하는 경기에서, 도약 후 $t$초 후 수면으로부터의 높이가 $h = -5t^2+4t+10$일 때, 선수가 가장 높이 올라갔을 때 수면으로부터의 높이를 구하시오. (단, $0 < t < 1.8$)

$h = -5t^2+4t+10 = -5\left(t-\dfrac{2}{5}\right)^2+10+\dfrac{4}{5}$
$= -5\left(t-\dfrac{2}{5}\right)^2+\dfrac{54}{5}$

$t = \dfrac{2}{5} = 0.4$초일 때 최댓값 $\dfrac{54}{5} = 10.8$m
답: $\dfrac{54}{5}$m (= 10.8m)

문제 3: 폭이 20cm인 양철판의 양 끝을 $x$cm씩 접어 단면이 직사각형인 물받이를 만들 때, 단면의 넓이 $S$가 최대일 때 $x$의 값을 구하시오.

밑면의 가로: $(20-2x)$cm, 높이: $x$cm
$S = x(20-2x) = -2x^2+20x = -2(x-5)^2+50$
$0 < x < 10$에서 $x = 5$일 때 최대
답: $x = 5$cm (최대 넓이: 50cm²)

스스로 확인하기

확인 1: 다음 주어진 $x$의 값의 범위에서 이차함수의 최댓값과 최솟값을 구하시오.

(1) $y=x^2-8x+3$ ($2 \leq x \leq 8$)    (2) $y=-x^2+6x+3$ ($0 \leq x \leq 5$)

(3) $y=x^2+4x+7$ ($1 \leq x \leq 4$)

(1) $y = (x-4)^2-13$, 꼭짓점 $(4, -13)$ → 범위 안
$f(2) = 4-16+3 = -9$, $f(4) = -13$, $f(8) = 64-64+3 = 3$
최댓값: $3$, 최솟값: $-13$

(2) $y = -(x-3)^2+12$, 꼭짓점 $(3, 12)$ → 범위 안
$f(0) = 3$, $f(3) = 12$, $f(5) = -25+30+3 = 8$
최댓값: $12$, 최솟값: $3$

(3) $y = (x+2)^2+3$, 꼭짓점 $(-2, 3)$ → 범위 밖
$f(1) = 1+4+7 = 12$, $f(4) = 16+16+7 = 39$
최댓값: $39$, 최솟값: $12$
답: (1) 최댓값 3, 최솟값 $-13$ (2) 최댓값 12, 최솟값 3 (3) 최댓값 39, 최솟값 12

확인 2: $-1 \leq x \leq 4$에서 이차함수 $y=2x^2-8x+k$의 최댓값이 11일 때, 최솟값을 구하시오. (단, $k$는 실수)

$y = 2(x-2)^2-8+k$, 꼭짓점 $(2, k-8)$ → 범위 안
$f(-1) = 2(9)-8+k = 10+k$
$f(4) = 2(4)-8+k = k$
$f(2) = k-8$ (최솟값)

최댓값은 $f(-1) = 10+k = 11$ → $k = 1$
최솟값 = $k-8 = 1-8 = -7$
답: 최솟값 $= -7$

확인 3: 이차함수 $f(x)=x^2+ax+3$이 $f(-1)=f(3)$을 만족시킬 때, $0 \leq x \leq 4$에서 $f(x)$의 최댓값과 최솟값을 구하시오. (단, $a$는 실수)

$f(-1) = 1-a+3 = 4-a$
$f(3) = 9+3a+3 = 12+3a$
$4-a = 12+3a$ → $-4a = 8$ → $a = -2$

$f(x) = x^2-2x+3 = (x-1)^2+2$
$f(0) = 3$, $f(1) = 2$ (최솟값), $f(4) = 11$ (최댓값)
답: 최댓값 $11$, 최솟값 $2$

핵심 공식 총정리

1. 복소수

$i^2 = -1$,   $a+bi$ ($a$: 실수부분, $b$: 허수부분)

켤레복소수: $\overline{a+bi} = a-bi$

$(a+bi)(a-bi) = a^2+b^2$

$i$의 거듭제곱: $i, -1, -i, 1$ 반복 (4개 주기)

$a > 0$일 때 $\sqrt{-a} = \sqrt{a}i$

2. 판별식

$ax^2+bx+c=0$에서 $D = b^2-4ac$

$D > 0$: 서로 다른 두 실근  |  $D = 0$: 중근  |  $D < 0$: 서로 다른 두 허근

$ax^2+2b'x+c=0$일 때 $\dfrac{D}{4} = b'^2-ac$

3. 근과 계수의 관계

$ax^2+bx+c=0$의 두 근 $\alpha$, $\beta$:

$\alpha+\beta = -\dfrac{b}{a}$,    $\alpha\beta = \dfrac{c}{a}$

두 수를 근으로 하는 이차방정식: $x^2-(\alpha+\beta)x+\alpha\beta = 0$

4. 이차함수와 이차방정식

$y=ax^2+bx+c$의 그래프와 $x$축의 교점 = $ax^2+bx+c=0$의 실근

그래프와 직선 $y=mx+n$: $ax^2+(b-m)x+(c-n)=0$의 판별식으로 판별

5. 이차함수의 최대, 최소

$f(x)=a(x-p)^2+q$, 범위 $\alpha \leq x \leq \beta$

꼭짓점 포함 여부 확인 → 양 끝점 + 꼭짓점 함숫값 비교

$a > 0$: 꼭짓점에서 최솟값  |  $a < 0$: 꼭짓점에서 최댓값

자주 쓰이는 변환 공식

$\alpha^2+\beta^2 = (\alpha+\beta)^2-2\alpha\beta$

$(\alpha-\beta)^2 = (\alpha+\beta)^2-4\alpha\beta$

$\dfrac{1}{\alpha}+\dfrac{1}{\beta} = \dfrac{\alpha+\beta}{\alpha\beta}$

지원아, 여기까지 다 읽었으면 방정식과 부등식은 이제 정복한 거야! 화이팅!
